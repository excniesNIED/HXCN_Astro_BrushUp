**本题为填空题，只需要算出结果后，在代码中使用输出语句将所填结果输出即可。**

已知大写字母 A 的 ASCII 码为 65，请问大写字母 L 的 ASCII 码是多少？

大写字母按顺序排列，A的ASCII码为65。L是第12个字母（A为1，B为2，...，L为12），因此计算其ASCII码为65 + (12 - 1) = 76。

```python
print(76)
```
